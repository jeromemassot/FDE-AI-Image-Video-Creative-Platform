# Creation of a video from an image
This is a short checklist of tasks to complete for creating a video from an image:

## 1. Source Image Selection
Description: Select the source image and identify its core subjects, composition, and overall mood.

## 2. AI Analysis of the image
Description: Use Gemini 2.5 Flash to generate a rich, detailed text description of the source image, 
capturing everything from textures and lighting to artistic style. In this application, this step
is done automatically when an image is uploaded in the image creation/annotation panel.

## 3. Video Prompt Redaction / Image annotation
Description: Review the AI-generated description of the source image. Modify it to steer the video creation toward 
your desired creative vision. You can also annotate the source image using the Annotation toolbox.

## 4. Video Generation
Description: Click first on the "Prepare for Video" button in the Image Creation/Annotation panel, and then 
click the "Generate Video" button to create the video from the prompt and the source image.
Remember that with annotated image, the video prompt is optional.

## 5. Selection & Iteration
Description: Generate multiple variations and adjust the prompt and repeat the 
generation step to explore different outcomes.

## 6. Finalize & Export
Description: Once you are satisfied with a generated video, download it on your computer from the embedded video player.
